SURGICAL GLOVE DETECTION USING COMPUTER VISION & DEEP LEARNING
***********************

Real-time detection of surgical gloves using a deep learning object detection model for medical safety and automation.

***********************
PROJECT OVERVIEW
***********************

This project focuses on detecting surgical gloves in images and live video streams using a trained object detection model.

It is designed for:
• Medical safety compliance  
• Industrial automation environments  
• Real-time computer vision systems  

The system identifies whether gloves are present and can be integrated into inspection pipelines.
